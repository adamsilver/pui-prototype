Guidance HMCTS online services for professionals

Find out how to create an account and use HM Courts and Tribunals (HMCTS) online services.

Published 20 August 2018
Last updated 22 August 2022
From: HM Courts and Tribunals Service

Getting started

You need to create an account before you can create or manage cases. You'll need to know your organisation's Solicitors Regulation Authority (SRA) number.

You can manage your account if you need to change any information later - for example if you move to a different office.

Creating new cases

Once you have an account for HMCTS online services, you can create a case online for:

  • divorce
  • financial remedy
  • probate
  • civil money claims

Managing existing cases

You can manage a case which you created or have been given access to. You'll be able to:

  • check the status of the case
  • manage user access - for example if you want to let a colleague work on it
  • upload documents
